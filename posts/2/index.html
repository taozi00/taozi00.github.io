二、上传jar包到服务器

2.1 用FileZilla可视化软件上传。

三、启动jar包

3.1 首先查看端口下的进程
[root@localhost demo]# netstat -lanp|grep
tcp 0 0 :::8888 :::* LISTEN 4601/java
1
2
3.2 杀掉该进程
[root@localhost demo]# kill -9 4601
1
3.3 启动jar包
[root@localhost demo]# nohup java -jar demo-1.0.1.jar >springboot.log 2>&1 &
